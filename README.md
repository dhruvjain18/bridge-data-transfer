# Bridge - Secure Data Transfer

**Live Website:** [https://bridgedata.onrender.com/](https://bridgedata.onrender.com/)

Bridge is a secure file transfer platform that lets you send files and messages directly to anyone via Telegram or WhatsApp or email - no sign-up, no hassle.

## Features
- **Send via Telegram:** Instant file transfer using a Telegram bot.
- **Send via WhatsApp:** Link your WhatsApp via QR code and securely push files to anyone.
- **End-to-End Privacy:** Files are instantly deleted from the server after delivery.
- **Schedule Transfers:** Pick a future date and time for files to be sent automatically.
- **File Support:** Send PDFs, docs, images, spreadsheets, ZIPs, and code files up to 50 MB.
