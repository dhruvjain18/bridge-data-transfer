💬 Help & FAQ

Getting Started

What is Bridge?

Bridge is a secure file transfer platform that lets you send files and messages directly to anyone via Telegram or WhatsApp — no email, no sign-up, no hassle.

How do I find my Telegram username?

Open Telegram → Settings → your username starts with @. Example: @dhruv_19

How do I register for the Telegram bot?

Search @File_Bridge_19_bot on Telegram and tap Start. You are now registered forever!

How do I access WhatsApp?

Click the WhatsApp tab, then scan the QR code with your own WhatsApp (Settings → Linked Devices → Link a Device). That's it — no phone verification needed! Your session is saved and will auto-reconnect.

Install Telegram

How to install Telegram on Android?

Open Google Play Store → search "Telegram" → Install → sign up with your phone number.

How to install Telegram on iPhone?

Open App Store → search "Telegram Messenger" → Get → register with your phone number.

How to use Telegram in the browser?

Visit web.telegram.org and enter your phone number to log in.

Features

Can I send to multiple people at once?

Yes! Separate usernames or phone numbers with commas. Example: @user1, @user2

What is "Schedule for later"?

Click 📅 Schedule for later, pick a future date and time, and your files will be sent automatically at that time.

What happens if my file exceeds 50 MB?

Bridge uses Auto-Zip — it compresses oversized files into a ZIP before sending. You'll see a yellow badge.

What file types are supported?

All file types — PDFs, docs, images, spreadsheets, ZIPs, code files, videos. Limit: 50 MB per file.

Can I send just a text message?

Yes! Leave the file zone empty and type your message. It will be sent as a regular text message.

What is the Transfer History?

Click 🕘 on the bottom-right to view all past transfers. History is stored locally in your browser.

Can I install Bridge as an app?

Yes! In Chrome, click the install icon in the address bar or go to Menu → "Install App".

Privacy & Security

Can others see my files?

No. Files go directly to the recipient's private chat. Files are deleted instantly from the server after delivery.

How does WhatsApp authentication work?

Each user links their own WhatsApp via QR scan — this is the only authentication needed. Your session is saved securely — no need to re-scan on reconnect. You can send messages to any phone number.

Is my data stored anywhere?

No. Files exist on the server only for seconds. Transfer history is stored only in your browser.
